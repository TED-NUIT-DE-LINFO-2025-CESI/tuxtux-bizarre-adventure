# Audio Assets

## BGM (Background Music)

Place these files in `bgm/`:

| Filename | Description | Duration |
|----------|-------------|----------|
| `intro.mp3` | Mysterious/ambient intro music | ~1-2min loop |
| `windows.mp3` | Corporate/synthetic music for Windows path | ~1-2min loop |
| `linux.mp3` | Uplifting/heroic music for Linux path | ~1-2min loop |
| `chaos.mp3` | Intense/glitchy music for chaos scenes | ~1-2min loop |
| `battle.mp3` | Epic battle music | ~2-3min loop |
| `victory.mp3` | Triumphant victory fanfare | ~30s-1min |

## SFX (Sound Effects)

Place these files in `sfx/`:

| Filename | Description | Duration |
|----------|-------------|----------|
| `blip.mp3` | Text appear sound | <0.5s |
| `hover.mp3` | Menu hover sound | <0.3s |
| `select.mp3` | Menu select/click sound | <0.5s |
| `attack-tux.mp3` | Tux attack sound | <1s |
| `attack-omega.mp3` | Omega attack sound | <1s |
| `hit.mp3` | Damage taken sound | <0.5s |
| `victory-fanfare.mp3` | Victory SFX | ~2-3s |

## Format

Prefer `.mp3` for compatibility. `.ogg` is also supported.
